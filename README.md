# Priority Queue using Binary Search Tree (Java)

This project implements a **Priority Queue** using a **Binary Search Tree (BST)** in Java.  
Students are stored based on their grades, where the student with the **highest grade** has the **highest priority**.

---

## Features

- Insert students into the priority queue (`Push`)
- Remove the student with the highest grade (`Pop`)
- Display all students using **Inorder Traversal**
- Count **comparisons** and **swaps**
- Interactive **menu** for user input or using predefined students

---

## Classes

1. **Student.java** → represents a student with name, ID, and grade  
2. **Node.java** → represents a node in the BST  
3. **PriorityQueueBST.java** → contains the priority queue logic  
4. **Main.java** → program execution with **interactive menu**
